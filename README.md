## 项目结构
- [jt-common](./jt-common): 工具 API
- [jt-manage](./jt-manage): 后端业务系统 (提供者)
- [jt-web](./jt-web): 前端 Web (消费者)
- [jt-sso](./jt-sso): 单点登录 (提供者)
- [jt-cart](./jt-cart): 购物车 (提供者)
- [jt-order](./jt-order): 购物车 (提供者)

```
├─jt
│  ├─jt-common
│  ├─jt-manage
│  ├─jt-web
│  ├─jt-sso
│  ├─jt-cart
│  └─jt-order
```
